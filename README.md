## ABOUT
### Project concept
The project is a geometry simulation program.

### Language
It is written in __C++__

### Libraries
    - Raylib
    - nlohmann/json

### Build

The project uses CMake for building.

#### How to build

Run in terminal

    `mkdir build && cd build`

    `cmake ..`

    `cmake --build . --config Release`

## INSTALL
Open your terminal and run:
